[RelayCommand]
private async Task IntegrateItems()
{
    // ...
    // Get IDs of items to integrate
    var itemIds = itemsToIntegrate.Select(i => i.Id).ToList();
    
    // Call BomIntegrationService to integrate into Sage
    bool success = await _bomIntegrationService.IntegrateNewItemsAsync(itemIds);
    
    // Reload items to get updated integration status
    await LoadItems();
    
    // Count results from reloaded data
    int successCount = itemsToIntegrate.Count(i => i.IsIntegrated);
    int failedCount = itemsToIntegrate.Count - successCount;
    
    // Enhanced error handling and user feedback

## ?? User Experience Flow

### Step-by-Step Integration Process

#### 1. **User Prepares Items**
```
1. User opens New Make Items view
2. Filters items to integrate
3. Edits required fields:
   - Item Description
   - Product Line (REQUIRED)
   - Product Type
   - Standard UOM
   - etc.
4. Clicks "Integrate" button
```

#### 2. **System Validates**
```
System checks:
? Items have Product Line set
? Items are not already integrated
? At least 1 item ready

If validation fails:
? Shows info message
? Prompts user to set Product Line
```

#### 3. **User Confirms**
```
Dialog appears:

?  using COM integration. Continue?       ?
?                                         ?
?         [ Yes ]        [ No ]           ?
???????????????????????????????????????????
```

#### 4. **System Integrates**
```
Progress:
???????????????????????????????????????????
?  Status: Integrating items into         ?
?         Sage 100...                     ?
?                                         ?
?  [?????????????????????] 15/20          ?
???????????????????????????????????????????

Background Process:
1. Load Sage settings
2. Initialize Sage session
   - Create ProvideX.Script COM object
   - Connect to Sage 100
   - Set user, company, date, module
3. Set program context (CI_ItemMaintenance_ui)
4. For each item:
   - Create CI_ItemCode_bus object
   - Set key (ItemCode)
   - Set field values
   - Write to Sage (nWrite)
   - Mark as integrated in database
   - Release COM object
5. Cleanup session
```
