## Duplicate Detection SQL Logic

```sql
/*
 * Main Query to Identify Duplicates
 * This query shows which imported BOMs already exist in Sage
 */
 
SELECT 
    ib.ParentItemCode as ImportedBillNo,
    ib.ComponentItemCode,
    b.BillNo as ExistingBillNo,
    ib.Status,
    CASE 
        WHEN b.BillNo IS NOT NULL 
        THEN 'DUPLICATE - Exists in Sage'
        ELSE 'NEW - Not in Sage'
    END as DuplicateStatus
FROM isBOMImportBills ib
LEFT JOIN BM_BillHeader b 
    ON ib.ParentItemCode = b.BillNo
WHERE ib.ParentItemCode IS NOT NULL
ORDER BY ib.ParentItemCode, ib.LineNumber;
```

## Real-World Example

### Scenario: Importing BOMs

**Excel File Contents:**
```
Row | ParentItemCode | ComponentItemCode | Quantity
1   | ASSEMBLY-101   | SCREW-M3         | 4
2   | ASSEMBLY-101   | WASHER-M3        | 4
3   | ASSEMBLY-101   | BRACKET-A        | 1
4   | NEW-ITEM-001   | BOLT-M5          | 2
5   | NEW-ITEM-001   | NUT-M5           | 2
```

**Database State:**
```sql
-- BM_BillHeader table
BillNo          | Description
----------------|------------------
ASSEMBLY-101    | Existing Assembly
ASSEMBLY-202    | Another Assembly
```

**After Import:**
```
ParentItemCode | ComponentItemCode | Status      | Reason
---------------|-------------------|-------------|------------------------
ASSEMBLY-101   | SCREW-M3         | Duplicate   | Exists in BM_BillHeader
ASSEMBLY-101   | WASHER-M3        | Duplicate   | Exists in BM_BillHeader
ASSEMBLY-101   | BRACKET-A        | Duplicate   | Exists in BM_BillHeader
NEW-ITEM-001   | BOLT-M5          | Validated   | New BOM, component exists
NEW-ITEM-001   | NUT-M5           | Validated   | New BOM, component exists
```

**Statistics:**
- Total Records: 5
- Duplicates: 3 (ASSEMBLY-101)
- Validated: 2 (NEW-ITEM-001)

## Key Takeaways

1. ? **ParentItemCode = BillNo** in BM_BillHeader
2. ? **One duplicate parent** = **ALL components marked duplicate**
3. ? **Duplicate check happens BEFORE component validation**
4. ? **Duplicates are NOT integrated** into Sage
5. ? **Clear validation messages** explain why duplicate
